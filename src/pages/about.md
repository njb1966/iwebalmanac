# About

Independent Web Almanac is written from the perspective of an experienced internet native with an infrastructure mindset.

This is not a personal diary. It is an editorial project about keeping the web useful, personal, and owned by the people who build and inhabit it.

No ads. No analytics. No third-party JavaScript. No external fonts. No tracking.
